1) 스펙·강화·장비

스펙: 캐릭터의 전투 능력 전반(스탯, 장비, 스킬, 링크/유니온 포함).

스펙업: 전투력을 올리는 모든 행동(강화, 교체, 세팅 변경 등).

종결 / 종결템: 해당 부위에서 사실상 최종 목표로 보는 장비(“이 이상은 손댈 이유 적다”).

준종결: 종결 바로 아래 단계. 비용 대비 효율이 좋아 많이 멈추는 구간.

템 / 템셋: 아이템 / 아이템 구성(장비 세팅).

세팅: 장비/스킬/스탯/하이퍼/어빌 등 전체 구성의 조합.

잠재(잠재능력): 장비에 붙는 추가 옵션(옵션 줄이 랜덤). 성능에 큰 영향.

에디(에디셔널): 잠재와 별개로 붙는 추가 옵션(보조 잠재 같은 개념).

줄(1줄/2줄/3줄): 잠재/에디 옵션 줄 개수 표현. “3줄”은 좋은 옵션이 3개 붙었다는 의미로 쓰임.

유효옵 / 유효: 실제 성능에 도움이 되는 옵션.

쓰레기옵 / 똥옵: 성능에 별 도움 없는 옵션.

강화(스타포스/스타): 장비를 별(★)로 강화해 스탯을 올리는 시스템.

작(주문서작/작업): 주문서를 사용해 장비에 스탯을 부여하는 강화 과정.

추옵(추가옵션): 장비에 기본으로 붙는 랜덤 추가 능력치. “추옵 좋다/나쁘다”로 성능 평가.

환불(환생의 불꽃): 추옵을 다시 뽑는 아이템/행위.

리턴 / 리턴작: 실패 위험이 있거나 비용이 큰 강화에서 되돌리는/복구하는 방식(맥락에 따라 다름).

큐브 / 큐빙: 잠재/에디 옵션을 다시 뽑는 아이템/행위.

가성비: 투자 대비 효율. “적은 돈으로 체감 상승이 큰 구간/방법”.

가챠(뽑기): 확률형 강화나 상자 구매를 뽑기처럼 부르는 표현.

2) 스탯·딜(대미지) 관련

주스탯(주스텟): 직업이 주로 사용하는 기본 능력치(STR/DEX/INT/LUK 중 하나).

부스탯: 주스탯을 보조하는 능력치(직업별로 의미가 달라짐).

공/마(공격력/마력): 물리/마법 공격 성능의 핵심 수치.

보공(보스 공격력): 보스에게 주는 피해 증가.

방무(방어율 무시): 보스의 방어율을 무시해 실제 피해를 높이는 수치.

크확(크리티컬 확률): 치명타가 발생할 확률.

크뎀(크리티컬 데미지): 치명타 피해 증가.

쿨감(쿨다운 감소): 스킬 재사용 대기시간 감소.

딜 / 화력: 실제로 넣는 피해량(“딜이 세다/약하다”).

딜러 / 서포터: 주 피해 담당 / 버프·유틸로 파티를 돕는 역할.

유틸: 생존·이동·보조·파티 기여(무적, 바인드, 해제, 보호막 등) 같은 편의/지원 요소.

극딜: 짧은 시간에 피해를 몰아 넣는 구간(버프를 다 켠 “한 방 타임”).

평딜: 극딜이 아닌 평상시 지속 피해.

딜압축: 극딜에 피해가 얼마나 잘 몰리는지(시간이 짧을수록 “압축이 좋다”).

딜사이클: 스킬을 어떤 순서/주기로 돌리면 최적 딜이 나오는지의 운영 방식.

환산(환산주스탯): 여러 스펙 요소를 하나의 수치로 환산해 비교하는 개념(“환산 몇만”). 계산기/기준에 따라 달라짐.

3) 컨텐츠·성장 체계

무릉: 무릉도장. 층수와 클리어 시간으로 기록 경쟁/스펙 판단을 함.

층수(몇층): 무릉에서 도달한 층. 스펙을 말할 때 “무릉 50층”처럼 사용.

커트라인: 어떤 기준을 “넘긴다”로 합의된 최소 조건(예: 무릉 50층이 기준이면 50층이 커트).

보스돌이: 여러 캐릭터로 보스를 반복 처치해 재화를 버는 방식.

주보/주간보스: 주 1회 가능 보스들.

일보/일일보스: 매일 가능한 보스들.

검마/세렌/칼로스/카링 등: 최상위 보스들을 줄여 부르는 말(보스 이름 축약).

해방: 특정 최종 단계(주로 제네시스 무기 관련)를 완료해 제약이 풀리는 상태를 말하는 경우가 많음.

링크(링크 스킬): 다른 캐릭터가 제공하는 공유 스킬(계정 단위 성장 요소).

유니온: 계정 단위 성장 컨텐츠(여러 캐릭터를 키워 효과 누적).

하이퍼스탯(하이퍼): 포인트를 찍어 스탯을 강화하는 성장 요소.

코어 / 5차 코어 / 코젬: 5차 스킬 강화 시스템 및 재료(“코어 젬스톤”).

헥사 / 6차 / 헥사코어: 6차 관련 강화 시스템을 통칭해 부르는 표현.

리멘/세르니움/아르크스…: 주요 사냥터/지역명을 줄여 부르는 경우가 많음(지역 자체가 메타 용어).

4) 직업·밸런스·메타 표현

메타: 현재 가장 효율적/유행하는 직업·세팅·플레이 방식의 흐름.

떡상: 갑자기 크게 좋아짐(주로 패치로 성능 상승).

떡락: 갑자기 크게 나빠짐(너프, 환경 변화).

너프/버프: 성능 하향/상향.

상향/하향: 버프/너프와 동일하게 쓰임.

리마스터: 직업 구조/스킬을 크게 개편하는 업데이트.

사기/사기캐: 지나치게 강하다고 느껴질 때 쓰는 과장 표현(실제 수치와 별개로 감정이 섞임).

고점/저점: 잘하면(조건 갖추면) 도달 가능한 최대 성능 / 조건이 안 좋을 때의 성능.

체급: 직업/세팅이 가진 “기본 성능 등급”을 뭉뚱그려 말함.

유리/불리: 특정 컨텐츠나 환경에서 성능이 좋다/나쁘다.

5) 파티·레이드·플레이

격수: 파티에서 주로 딜을 넣는 사람(딜러 포지션).

시너지: 파티원이 주는 버프/디버프가 서로 합쳐져 화력이 올라가는 효과.

바인드: 보스를 일정 시간 묶어 두는 상태이상(극딜 타이밍에 중요).

무적기: 일정 시간 피해를 받지 않는 스킬.

생존기: 회복/보호막/피해 감소/회피 등 생존에 도움 되는 수단.

트라이: 공략 연습 시도(“몇 트라이 했음”).

클리어/격파: 보스를 잡음.

리트/리트라이: 실패 후 다시 시도.

딜찍: 패턴을 제대로 보기 전에 딜로 눌러 잡는 것(스펙으로 “찍어 누름”).

숙련/숙코: 숙련자 / 숙련자인 척하는 사람(커뮤니티에서 흔한 표현).

6) 경제·거래·재화

메소: 게임 내 기본 화폐.

메획/드획: 메소 획득량/드롭률 증가(사냥 효율 세팅에서 자주 언급).

경뿌: 경험치 버프(“경험치 뿌리기”에서 온 약어로 쓰이는 경우가 흔함).

시드링(링): 특정 반지(특히 리스크/리턴/웨폰퍼프 등) 기반 딜 상승 장치들을 통칭.

물통/매물: 거래소에 올라온 물건/판매 물량.

시세: 시장 가격.

현질/과금: 현금 결제.

7) 자주 헷갈리는 표현 (해석 팁)

“스펙 대비”: 같은 투자/스펙 구간에서 성능이 좋은가를 말함(직업/세팅 비교 맥락).

“체감”: 수치보다 플레이 느낌(편의성, 조작 난이도, 패턴 대응)을 의미.

“구조가 좋다/나쁘다”: 스킬 메커니즘(딜사이클, 무적/바인드 보유, 이동기 등)을 평가.
